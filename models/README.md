# Models Directory

This directory contains data models and schemas for the J.A.R.V.I.S application.

## Purpose

Data models define the structure of data used throughout the application, including:
- Voice command structures
- User preferences and settings
- API response models
- Local database schemas

## Example Models

- `voice_command.dart` - Voice command data structure
- `user_preferences.dart` - User settings and preferences
- `assistant_response.dart` - Response from the assistant
- `command_history.dart` - Historical command data
